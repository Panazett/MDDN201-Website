Wellingtons Internet Infrastructure

What is Infrastructure?

the basic physical and organizational structures and facilities (e.g. buildings, roads, power supplies) needed for the operation of a society or enterprise - Google

Why is Infrastructure Needed?

Infrastructure provides the fundamental elements need to form and maintain complex systems, an example of this, is the internet

Found Around Wellington

Internet related hardware, and applications that utilize this hardware can be found scattered around Wellington

Hardware

The main components of internet hardware would include items such as routers and antennas

Applications

Applications that use the network would include items such as security systems and parking meters

Hardware

Ethernet Cables

Ethernet cables, also classed as Cat-5 cables are used to transfer broadband signals to, and from connected devices.

Ethernet Port

Ethernet ports are the end points between a connection between two connected devices by the means of an ethernet cable.

Phone Line

Phone Lines we're once the main method of data transfer before the standard of ADSL data transfer was developed.

Routers

Routers are used to traffic the network data flow. Routers are used to connect multiple devices to a single network.

Wireless Transimitters

Wireless transmitters are used to broadcast wireless frequencies, allowing devices to access the internet.

Applications

Security Systems

Security systems take advantage of local networks ability to communicate, allowing communication between multiple security orientated devices.

Security Cameras

Security cameras use networks to transfer recorded data and imagery to devices capable of mass storage.
